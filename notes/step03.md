# Step 3 - Arrays and Introduction to Linked Lists

## What I learned

- Arrays provide fast access by index (`O(1)`).
- Searching in an array is usually linear (`O(n)`).
- Inserting at the beginning of an array is slow because existing elements must be shifted.
- `append()` is usually fast because Python lists often have extra allocated space.
- When that space is full, Python allocates a larger memory block and copies the elements.
- Linked Lists were created to solve the slow insertion problem of arrays.
- In a Linked List, elements are connected by references instead of being stored next to each other.
- Accessing an element by index is fast in arrays but slow in linked lists.
